# OpenChrome

原文: <<https://www.freedesktop.org/wiki/Openchrome>

## 关于 Openchrome

Openchrome 项目致力于提供和支持完全免费的开源驱动程序，充分利用威盛 UniChrome、UniChrome Pro 和 Chrome9 集成图形处理器的威盛芯片组的硬件加速。
